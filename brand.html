Parte A — Identidad base / 01

01 Concepto

El trabajo full stack tiene capas — y esas capas se muestran, no se esconden.

"Exploded view": interfaz, API y datos apilados como motivo visual constante — franjas de fondo y un "layer badge" que acompaña cada proyecto para mostrar de qué está hecho. Dirección deliberadamente distinta a la terminal oscura genérica o al cuaderno notion-style: mostrar lo que soy, alguien que entrega la pila completa.

Reglas de consistencia

12 Consistencia entre piezas

  • El código de color de capas (lima=UI, magenta=API, cian=datos) nunca se reasigna, en ningún material ni plataforma.
  • Los roles tipográficos son fijos: Instrument Serif itálica nunca es texto de interfaz, DM Mono nunca es cuerpo largo.
  • El layer badge mantiene el mismo lugar y tamaño en toda portada donde aparezca.
  • Un mismo contenido se adapta entre plataformas sin que la identidad visual cambie según la red de destino.
